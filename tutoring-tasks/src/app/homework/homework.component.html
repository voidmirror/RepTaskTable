Массив - тип данных, представляющий собой несколько элементов одного типа, собранных под одним именем.

Матрица - двумерный массив, графически представляется в виде координатной сетки с началом координат в левом верхнем углу.
Координаты элемента матрицы M принято обозначать M[i][j], где i - координата строки, j - координата столбца.
Главная диагональ матрицы - диагональ с направлением из левого верхнего угла в правый нижний.
Свойство всех элементов M[i][j] главной диагонали: i == j.
Обход матрицы производится в помощью вложенного цикла for:
    for i in range(len(M)): # Количество строк в матрице (длина массива массивов)
        for j in range(len(M[i])): # Длина одной взятой i-той строки матрицы
            ...
